InkedMark

Handwriting that lives inside your Obsidian notes — pressure-aware ink, fused with markdown, searchable and graphable like everything else in your vault.

View on GitHub Install

Write by hand, in markdown

Handwriting is a first-class block inside ordinary .md notes — a dedicated *.ink.md canvas or an inline block. Type where typing is better, write where writing is better, in the same note.

Searchable & graphable

Every ink note carries a text layer that is real markdown — transcription, [[links]], #tags. Obsidian's search, graph, backlinks, and Quick Switcher index your handwriting with no custom engine.

Built for Apple Pencil

GoodNotes-class ink feel within the webview: pressure-variable strokes via perfect-freehand, full-rate capture with coalesced events, a wet/dry canvas split, and low-latency rendering. Local-first, single-file, sync-friendly.

Install

  • Community Plugins: search for “InkedMark” (once accepted).
  • BRAT (beta): add pcrausaz/obsidian-inkedmark in the BRAT plugin.
  • Manual: copy main.js, manifest.json, and styles.css from a release into <vault>/.obsidian/plugins/inkedmark/.

iPad: turn off Settings → Apple Pencil → Scribble so fast strokes aren't intercepted by iPadOS before InkedMark sees them.
